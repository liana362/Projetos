Technical Skills


Liana Gonzaga possui uma sólida base em desenvolvimento web, com foco em tecnologias front-end e back-end. No front-end, ela domina HTML5, CSS3 e JavaScript (ES6+), aplicando suas habilidades para criar interfaces interativas, acessíveis e responsivas. Seu conhecimento em frameworks como React.js e Vue.js permite o desenvolvimento de aplicações modernas e dinâmicas, enquanto ferramentas como Bootstrap garantem agilidade na criação de layouts otimizados. No back-end, Liana utiliza Node.js e Express.js para desenvolver APIs RESTful robustas e escaláveis. Além disso, possui experiência no gerenciamento de bancos de dados, como MongoDB e MySQL, garantindo a estruturação eficiente de dados tanto em ambientes NoSQL quanto relacionais.
